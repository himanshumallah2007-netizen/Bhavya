# For Bhavya ❤️

A sweet, romantic, interactive digital proposal and letter designed specially for Bhavya.

## Features
- **Falling Rose Petals**: A beautiful, 7-second shower of custom rose petals greets her upon entry.
- **Scrapbook Letter**: A heartfelt, beautiful scrapbook letter with a 3D animated envelope interaction.
- **The Proposal**: A final screen asking the big question with interactive buttons.
- **Success Screen**: A celebratory screen for when she says Yes!
- **Aesthetic**: A custom pink scrapbook theme with adorable Hello Kitty, Kuromi, and Bear elements perfectly curated for her.

## How to View
Simply double-click on `index.html` to open it in your favorite web browser and experience the magic!

## Files Overview
- `index.html`: The main structure of the website.
- `style.css`: The beautiful styles, animations, 3D elements, and CSS magic that make it look incredible.
- `script.js`: The interactivity, logic, and falling rose petal generation.
- Various `.png` and `.jpg` image files you've handpicked for the experience.
